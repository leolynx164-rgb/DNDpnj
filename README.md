## Configuration

L’application fonctionne via l’API OpenRouter.

Dans **Paramètres**, entrez le modèle d’IA (par exemple : `openai/gpt-oss-20b:free` suffit) ainsi que votre clé API, récupérable sur https://openrouter.ai/

Le endpoint OpenRouter est généralement :
**https://openrouter.ai/api/v1/chat/completions**
